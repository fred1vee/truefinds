# TrueFinds — Project Context

## Project

TrueFinds — это современная платформа для обучения заказу товаров с Pinduoduo и последующего ресейла в Азербайджане.

Проект разрабатывается как MVP с последующим переходом на полноценный production backend.

---

## Goal

Создать лучший сервис в Азербайджане для:

- поиска выгодных товаров;
- обучения заказу товаров;
- удобного ресейла;
- использования полезных инструментов.
